## 概要
条件を満たす区間の個数を数えたり。
左端を固定したとき、どこまで右に行けるか。答えに単調性があるときに使える。

- whileするとき、`r < n` を忘れない
- 条件を満たしているならrを増やすので、whileを抜けたとき、$[l,r)$ が条件を満たす区間
- lがずれるとき、管理してる値があるなら値を更新する

## 計算量
$$
O(N)
$$
## 実装
問題によって複数パターンがありそう

### 区間の総和がK以下
```python
ans = 0
sum_ = 0

r = 0
for l in range(n):
	while r < n and sum_ <= k:
		s += a[r]
		r += 1
	ans += r - l
	s -= a[l]
```

### 隣り合う数字の関係をみるとき
- 連続して同じ文字が表れないという条件で
`r = max(r, l+1)`大事

```python
ans = 0

r = 1
for l in range(n):
	r = max(r, l+1)
	while r < n and a[r-1] != a[r]:
		r += 1
	ans += r - l
```
