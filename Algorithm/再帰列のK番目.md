## 問題例
$i$ 番目の列 $S_i$ が漸化式になっていて、前の列の結合で表されるようなやつ。
下の例だと、先頭からある地点までの文字列が $S_i$ になっている感じ
```
   s[0]
|---|    s[1]
|---------| s[2]
|------------|
```

## 解
$S_i=S_{i-1}+S_{i-2}$ だとする。
$S_i$ の $k$ 番目の文字を、$f(i, k)$ とすると、
- もし $k\lt |S_{i-1}|$ なら、 $f(i-1, k)$
- そうでないなら、 $f(i-2,k-|S_{i-1}|)$
