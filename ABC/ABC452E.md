## とにかくiかjを固定しよう

### iを固定した場合
$i$ を固定したとき、 $i\bmod j$ がどうなるか考えよう
```python
# i=1: 0 1 1 1 1 1
# i=2: 0 0 2 2 2 2
# i=3: 0 1 0 3 3 3
# i=4: 0 0 1 0 4 4
# i=5: 0 1 2 1 0 5
```
縦に見ると、周期的になっているのが分かるだろう。
とりあえず基本は1ずつ増えると思っていいだろう。
これに $B_j$ がかかるので、実際には $i$ が1増えるごとに、 $sum(B)$ が1増える。

ここで、0に戻る場合を考えよう。よく見ると、周期はjになっている。
つまり、各列の0に戻る回数は $\frac{n}{j}$ 回である。したがって、合計で、
$\frac{n}{1}+\frac{n}{2}+...=O(N\log N)$ 回個別に0にすればいい。

具体的には $i$ を増やしていったとき、 $i$ の約数である $j$ について、 $-jB_j$ すればいい

**0にする回数が調和級数であることを思いつくのが1番難しいかもしれない**


### jを固定した場合
```python
# j=1: 0 0 0 0 0 0
# j=2: 1 0 1 0 1 0
# j=3: 1 2 0 1 2 0
# j=4: 1 2 3 0 1 2
# j=5: 1 2 3 4 0 1
```
こっちはさっきの転置になっている。
