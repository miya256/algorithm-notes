i番目まで白で、それより右を黒にする、最小コストという状態を持つ

遷移で、自分より右側のminをとる必要があるが、後ろからやれば累積minできる

コードの一部にこれがあるが、i行目について、j列目まで白にする場合の塗り替える個数
最初に全部黒にする場合をいれておき、
もし、"."だったら塗らなくてよくなるので、-1
もし"#"だったら、"."に塗り替える必要があるので+1
しているという感じ
```pythob
c = [s[i].count(".")]
for j in range(n):
	if s[i][j] == ".":
		c.append(c[-1]-1)
	else:
		c.append(c[-1]+1)
```
