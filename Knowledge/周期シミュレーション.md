## 問題
ある周期で数字が増減して、初めてある一定の値以上になるのは何周目の何番目かみたいな
[[#もし開始位置が異なる場合]]
## 初めて $t$ より小さくなる時点
初期値を $S$ 、$i$ 番目では $d_i$ 増加するとする。周期は $T$

### $\sum_{i=0}^{n-1}d_i\ge 0$ の場合
最初の周期で $t$ より小さくならないなら、増加していくだけなので、答えなし。
$S+\sum_{i=0}^{j}d_i\lt t$ になる最小の $j$ を求める（累積和をセグ木にいれてmax_rightとか）。1周目の $j$ が答え
### $\sum_{i=0}^{n-1}d_i\lt 0$ の場合
$m=\min_j \sum_{i=0}^{j}d_i$ 、つまり、$m$ を $d$ の累積和の最小値とすると、
$S+k\sum_{i=0}^{n-1}d_i + m\lt t$ を満たす最小の $k$ について、$k+1$ 周目で $t$ より小さくなる。
$$
\begin{align*}
S+k\sum_{i=0}^{n-1}d_i + m&\lt t\\
-k\sum_{i=0}^{n-1}d_i&\gt S+m-t\\
k&\gt \frac{S+m-t}{-\sum_{i=0}^{n-1}d_i}\\
k&= \left\lfloor\frac{S+m-t}{-\sum_{i=0}^{n-1}d_i}\right\rfloor+1
\end{align*}
$$

あとは $k$ をいれて、$S+k\sum_{i=0}^{n-1}d_i + \sum_{i=0}^{j}d_i\lt t$ になる最小の $j$ を求める。
式変形すると、$\sum_{i=0}^{j}d_i\lt t-S-k\sum_{i=0}^{n-1}d_i$ になる最小の $j$

---
## 初めて $t$ 以上になる時点
初期値を $S$ 、$i$ 番目では $d_i$ 増加するとする。周期は $T$

### $\sum_{i=0}^{n-1}d_i\le 0$ の場合
最初の周期で $t$ 以上にならないなら、減少していくだけなので、答えなし。
$S+\sum_{i=0}^{j}d_i\ge t$ になる最小の $j$ を求める（累積和をセグ木にいれてmax_rightとか）。1周目の $j$ が答え
### $\sum_{i=0}^{n-1}d_i\gt 0$ の場合
$m=\max_j \sum_{i=0}^{j}d_i$ 、つまり、$m$ を $d$ の累積和の最大値とすると、
$S+k\sum_{i=0}^{n-1}d_i + m\ge t$ を満たす最小の $k$ について、$k+1$ 周目で $t$ 以上になる。
$$
\begin{align*}
S+k\sum_{i=0}^{n-1}d_i + m&\ge t\\
k&\ge \frac{t-S-m}{\sum_{i=0}^{n-1}d_i}\\
k&= \left\lceil\frac{t-S-m}{\sum_{i=0}^{n-1}d_i}\right\rceil
\end{align*}
$$

あとは $k$ をいれて、$S+k\sum_{i=0}^{n-1}d_i + \sum_{i=0}^{j}d_i\ge t$ になる最小の $j$ を求める。
式変形すると、$\sum_{i=0}^{j}d_i\ge t-S-k\sum_{i=0}^{n-1}d_i$ になる最小の $j$

## もし開始位置が異なる場合
リストを2つつなげて、累積和を出しておく。で、開始地点から $T$ だけ先までの範囲だけ使って同じことをすれば行ける。セグ木で二分探索するときに範囲を超える可能性があるので、はみ出てないか判定。

### 結構苦戦したので添え字の部分とか書いときます
- 2つ繋げた累積和
```python
d = [0] + d + d
for i in range(len(d)-1):
    d[i+1] += d[i]
```

- `l` は0-indexed。`l` 番目から開始するということ
```python
sum_d = d[n+l] - d[l]
offset = d[l]
```

- sum_d >=0のほう
	- 先頭は0だから、l+1からだかんな
	- 二分探索のときのoffsetは足すんだぞ
	- n+l+1超えたとき場合分けしろよ
- elseのほう
	- prodのときのoffsetは引くんだぞ
	- sum_dは負だから、除算きをつけろよ
	- **kが負になる可能性あるから、max(0)しろよ**
```python
if sum_d >= 0:
	i = seg.max_right(l+1, lambda x: x >= -s + offset)
    if i >= n+l+1:
        print(0)
        continue
    print(i-l)
else:
	m = seg.prod(l+1, n+l+1)-offset
	k = max(0, (s+m) // (-sum_d)+1)
	i = seg.max_right(l+1, lambda x: x >= -s-k*sum_d+offset)
	print(k*n + i-l)
```
